# 🤖 AI Discord Assistance ChatBot

Intelligent Discord bot powered by **OpenRouter (free models)** and **Tavily** web search.  
Add it to your server and get **smart, up-to-date answers** in any channel.  

[![Add to Discord](https://img.shields.io/badge/➕-Invite%20Bot-7289DA?style=for-the-badge&logo=discord&logoColor=white)](https://discord.com/oauth2/authorize?client_id=1482288105863118929&scope=bot&permissions=68608)  

---

### ✨ Features
- **Natural chat** – AI answers messages without needing `!ai` every time  
- **Commands** – `!ai`, `!web`, `!ping` for structured interactions  
- **Live internet search** – `!web` fetches fresh data from Tavily  
- **Free & auto-updated models** – Always uses OpenRouter’s current free model  
- **Rate limits & safety** – Per-user cooldowns and error handling  
- **Logging** – All activity logged in `logs/bot.log` for easy debugging
